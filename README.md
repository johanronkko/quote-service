# Quote Service

Provides external customers with an API to send shipment information to Sendify, and for us to let them know how much the shipment will cost. 

## Project Structure

A lot of the boilerplate code and the project structure is inspired by [ardanlabs](https://github.com/ardanlabs/service/).
